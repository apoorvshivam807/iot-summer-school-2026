| Parameter | Microcontroller (Arduino UNO) | Microprocessor (Raspberry Pi) |
| :--- | :--- | :--- |
| **CPU Speed** | [cite_start]16 MHz [cite: 91] | [cite_start]1–2+ GHz (depends on Raspberry Pi model) [cite: 91] |
| **On-chip RAM** | [cite_start]2 KB SRAM [cite: 91] | [cite_start]Typically 1 GB–8 GB external RAM [cite: 91] |
| **On-chip Flash** | [cite_start]32 KB Flash Memory [cite: 91] | [cite_start]No on-chip flash; uses a microSD card for storage [cite: 91] |
| **Operating System** | [cite_start]No operating system (runs a single program directly) [cite: 91] | [cite_start]Runs an operating system such as Raspberry Pi OS (Linux) [cite: 91] |
| **Typical Use Case** | [cite_start]Embedded systems, sensor interfacing, robotics, IoT control [cite: 91] | [cite_start]General-purpose computing, multimedia, AI, IoT gateway, web server [cite: 91] |
| **Power Consumption** | [cite_start]Low (typically around 50–70 mA at 5 V) [cite: 91] | [cite_start]Higher (typically 2–7 W depending on the model and workload) [cite: 91] |
