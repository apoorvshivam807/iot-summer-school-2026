# Q13. Arduino Pin Types Explanation & IoT Use Cases

Below is the technical breakdown of the hardware pin allocations available on the Arduino UNO board, detailing their operation, functional examples, and real-world Internet of Things (IoT) deployment use cases.

| Pin Type | Explanation | Example | Real IoT Use Case |
| :--- | :--- | :--- | :--- |
| **Digital Input** | Reads only two distinct binary states: HIGH (1) or LOW (0) by checking voltage thresholds. | Push button or PIR motion sensor connected to Pin 2. | Detecting structural intrusion or motion for an automated smart security system. |
| **Digital Output**| Sends HIGH (5V) or LOW (0V) signals to drive or trigger external electronic components. | LED connected to Pin 13 or an isolated relay module trigger line. | Turning a mechanical power relay ON/OFF to control standard high-voltage home appliances. |
| **Analog Input** | Reads continuously variable analog voltage values (0–5 V) and converts them into digital integers (0–1023) using a 10-bit Successive Approximation ADC. | Potentiometer or resistive probe interface connected to Pin A0. | Monitoring volumetric soil moisture levels or localized ambient light intensity parameters in smart agriculture. |
| **PWM Output** | Produces Pulse Width Modulation square-wave signals to simulate a variable analog output by altering the Duty Cycle. Available on pins 3, 5, 6, 9, 10, and 11. | Fading an indicator LED smoothly or setting the precise rotation angle of a servo motor. | Adjusting the brightness of smart street lights or dynamically controlling DC fan motor speeds based on environmental heat. |
| **I2C / SPI Pins** | Dedicated hardware communication bus pins used to transfer structured data frames to sensors and modules. I2C uses SDA (A4) and SCL (A5); SPI uses pins 10, 11, 12, and 13. | OLED display screens, RTC real-time clock modules, SD card breakout shields, or Wi-Fi network coprocessors. | Connecting multiple specialized telemetry sensors and digital displays together on a shared bus in an environmental monitoring system. |
